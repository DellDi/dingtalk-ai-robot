## [1.4.0] - 2025-07-02

### 🚀 新特性
* **SQL 数据库查询代理**
  - 引入 `SQLTeamAgent`，采用 AutoGen 多智能体（Translator/Executor/Formatter）协作完成 “自然语言 → SQL → Markdown” 全流程。

### 🛠️ 技术改进
* 重构 `db_utils.get_conn()`：统一建表、注册注释逻辑。

* README 增加“本地数据库查询与分析”核心功能及元数据表说明。
